# How To Use

# 1) Generate programs through typing "make" in the command line

# 2) Use programs like this:
	./(factors|primefactors|factorial_factors).c [Natural Number]
	./factors 29
	./factors 329849
	./primefactors 2892
	./primfactors 2392i4248
	./factorial_factors 24244
	./factorial_factors 242

# 3) Enjoy the output!

# 4) Remove everything by make clean
